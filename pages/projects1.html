Capstone Project 1

HR Attrition

Baby steps! For the first module, we delve deeper into the magnificent Microsoft Excel. We covered Excel functions, Power Query, What-If analysis function including correlation and linear regression. Using the skills picked up, we had to create an interactive dashboard which will then be presented to the stakeholders - behold the trainers/mentors and fellow trainees! :)

Found an interesting dataset in the mighty Kaggle site about HR attrition.

StaffÂ attritionÂ refers to the loss of employees through a natural process, such as retirement, resignation, elimination of a position, personal health, or other similar reasons. WithÂ attrition, an employer will not fill the vacancy left by the former employee.

I was interested to identify the factors that encourage employees to leave the company. By identifying these factors, I could see if it was the internal or external factors that are the main attributes of cause. Therefore, the Management and HR team can work together and find possibilities to tweak its policies accordingly in order to retain its employees.

  • Business need: To identify which factors lead to employees leaving
  • Tool used: MS Excel
  • Data file type: CSV file
  • Source: HR Attrition Dataset (Kaggle)

Using the guide framework as advised by NTUC (check out my previous post here!), data cleaning was done using a great tool in MS Excel called Power Query (powerful tool). I removed duplicate rows, whitespaces, redundant columns, input missing values wherever possible and ensuring that the formattings are correct. After which, data transformation is done on the cleaned dataset like finding the total amount of entries.

Once completed, the dataset is loaded back into Excel and Exploratory Data Analysis (EDA) can begin! Pivot table function was mainly used to visualize the data and see for any interesting insights.
These insights were then further used to create an interactive dashboard in Excel using slicers and filters.
