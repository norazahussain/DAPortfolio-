Data Analytic Project Work Flow

Capstone Project Outline

Below is the typical Data Analytic guideline process workflow provided by NTUC Learning Hub. As a new entrant to the data analytics world, these workflow definitely helped and provide a starting point for me and my fellow trainees to gain better understanding.

Scope of Work

The main aim of the capstone project is to exhibit the five stages of the data lifecycle through the use of the tool: MS Excel. Participants, as part of the capstone project, will have to exhibit the following:

  • Data collection from various sources
  • Business needs gathering
  • Story building in relation to business needs
  • Data Preparation
    • ETL
    • Data Cleaning
    • Data Transformation
  • Exploratory Data Analysis using Excel/Power BI/Python/PPT
  • Creating Interactive Dashboard and Visualization using Excel/ Power BI/Python Dashboards
  • Communicating insights derived based on decided business needs
  • Presenting to the stakeholders

For each stage:

Data collection from various sources
Firstly, a domain of choice can be chosen on which analysis needs to be done. Areas could be sales, marketing, finance, retail, healthcare, or others. Data can be collected from various sources on the web such as:

Public Data Sources:

  • data.gov.sg
  • www.data.gov
  • www.data.world
  • AWS public datasets
  • Google public datasets
  • Socrata open data
  • Kaggle
  • Others
  • DataScience Free Data Set Links
  • APIs provided by various websites
  • Data from your organization (if permitted to use)

    • Business needs gathering
    This is the stage whereby the learner is to decide on a convincing business need which can be used for the chosen dataset. The learner is to discuss with the mentor on their ideas for the business needs. If the learner has a real-world scenario with a real business or company, he/she is encouraged to speak with the mentor about it.

    Story building
    After gathering the business needs, the next task is to build a storyline behind the data and the business needs. This exercise is to be iterated and refined based on the business needs.

    Creating tables and Loading data into Excel
    With the knowledge of Excel gained in the previous courses, import the data into Excel and create the required tables.

    Data Preparation
    After creating all the required tables and loading in the raw data, perform data cleaning and transformation operations. You could:

    • Clean unwanted data
    • Create and derive new columns
    • Remove anomalies and outliers in the data
    • Remove inconsistencies
    • And others.
    Exploratory Data Analysis using Excel
    Using the methods learnt during the Excel course, perform analysis on the data to derive useful insights. Make use of different excel tables, pivot charts and data summarization techniques.

    Creating Interactive Dashboard and Visualization using Excel Dashboards
    Once all the data has been prepared in Excel, import the data into Excel directly from SQL. Create required the pivot tables and charts based on the business needs and the story which has been defined, in the form of dashboards which help in visualizing the data and also in drawing useful insights. Create the dashboards with the intention of drilling down further into the data. Exposing the insights which you have found to help you tell a story to the stakeholders.

    Presenting to the stakeholders
    Once you have created all the required reports and dashboards, you would be presenting the same to the stakeholders (mentors/trainers). Present your dashboards in a way which will help you drive the presentation towards the ‚Äúaha‚Äù moment.
    The mentors and trainers will guide you through the whole process. Discuss every stage with your mentors and trainers to understand whether you are proceeding in the right direction or not. Put your creative thinking caps on and use the project to exhibit all your skills, like in a real organization!
