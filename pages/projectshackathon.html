Hackathon!

Power BI using Loan dataset

We had our first taste of hackathon!

Our mentors/trainers thought it would be an exciting experience for us to participate in hackathon. As we were still beginners, the hackathon is done amongst our training class randomly chosen into groups of 3 or 4. We were povided with the loan dataset from our mentors and had about 6 hours to come up with insights on the data which will be used to create BI report. :)

It was definitely a stressful yet an exciting experience but one where I would participate again in near future.

  • Business need: To generate new potential customers
  • Tool used: Power BI
  • Data file type: CSV file
  • Source: Loan Dataset (Kaggle)
