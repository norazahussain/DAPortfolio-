Capstone Project 2

What makes a movie successful?

For the second Capstone Project, we had to do both the data preparation(data cleaning & transformation) and EDA in MSSQL. After which, load the insights back into MS Excel and using the pivot tables to create visualizations of the analysis into an interactive dashboard (just like in previous capstone).

Since I started earning my keep, I was hooked into catching movies in the cinemas, watching them once they are released. At times, I wondered which type of movies are actually reeling in profits. Imagine if I was a producer/director, I would definitely be interested to know what is the formula to make a successful movie. Once again I found 2 datasets from the mighty Kaggle site; IMDB data and MUBI streaming platform data. Using data from MUBI allows me to find out which movies are popular amongst its customers and merging it with IMDF dataset allows me to extract the budget and revenue of the movies.

  • Business need: What makes a movie successful?
  • Tool used: MS SQL and MS Excel
  • Data file type: CSV files
  • Source:
    1. Maven Market (Kaggle)
    2. IMDB Dataset (Kaggle)

Data preparation
Data cleaning and transformation were done in MS SQL. Data cleaning includes removing duplicate rows of movie titles and rows with empty values in the budget or revenue columns and ensuring that the date/currency formattings are correct. After which, data transformation was done on the cleaned dataset.

Next, EDA was also done in MS SQL! New tables were made for the relevant insights found. Then they were exported back to MS Excel sheets to be visualized into an interactive report.
