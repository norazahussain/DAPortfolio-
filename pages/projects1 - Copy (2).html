Capstone Project 1

HR Attrition

We started from baby steps. For the first module, we delve deeper into the magnificent Microsoft Excel. We covered Excel functions, Power Query, What-If analysis function including correlation and linear regression. Using the skills picked up, we had to create an interactive dashboard which will then be presented to the stakeholders - behold the trainers/mentors and fellow trainees! :)
Found an interesting dataset in the mighty Kaggle site about HR attrition. I was interested to identify the factors that encourage employees to quit. By identifying these factors, we could see if it was the internal or external factors that are the main attributes of cause. Therefore, the Management and HR team can work together and find possibilities to tweak its policies accordingly in order to retain its employees.

  • Tool used: MS Excel
  • Data file type: CSV file
  • Business need: To identify which factors lead to employees leaving

First step after downloading the CSV file: Data preparation
Data preparation includes ETL, data cleaning and data transformation Data cleaning was done using a great tool in MS Excel called Power Query (powerful tool). Data cleaning includes removing duplicate rows, input missing values wherever possible and ensuring that the formattings are correct. After which, data transformation is done on the cleaned dataset.

Once completed, the dataset is loaded back into Excel and Exploratory Data Analysis (EDA) can begin! Pivot table function was used to visualize the data and see for any interesting insights.
These insights were then further used to create an interactive dashboard in Excel using slicers and filters.
